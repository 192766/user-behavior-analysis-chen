# 数据分析项目作品集

## 项目一：电商用户价值分析与流失预测

### 简介
基于英国电商数据集（Online Retail II），使用 RFM 模型对用户分层，构建逻辑回归流失预测模型，并利用 Power BI 制作交互式看板。

### 核心结论
- 前20%的高价值用户贡献了 **66.3%** 的销售额。
- 高价值用户平均购买 12.07 次，普通用户仅 2.30 次。
- 流失用户（>90天未购买）占比 33.3%，其平均消费金额仅为未流失用户的 1/7。
- 流失预测模型 AUC = 0.97，精确率 100%，可精准召回风险用户。

### 文件
- `电商用户价值分析.ipynb`：完整 Python 代码（数据清洗、RFM、逻辑回归、可视化）
- `电商用户价值分析与运营策略报告.md`：详细分析报告（含图表、业务建议）
- `dashboard/` 或截图：Power BI 看板截图（用户分层、流失分析等）

## 项目二：优惠券效果归因与渠道价值评估

### 简介
分析 50 万条模拟电商数据，评估优惠券策略的有效性及不同推荐渠道的价值。

### 核心结论
- 优惠券使用次数对购买数量无显著影响（p=0.901，系数≈0），当前策略无效。
- 各推荐渠道的转化率、购买数量、用户收入高度接近（差异<1%），业务上无显著区别。

### 文件
- `优惠券效果归因.ipynb`：Python 代码（回归、ANOVA、热力图、柱状图）
- `优惠券效果归因与渠道价值评估报告.md`：分析报告
