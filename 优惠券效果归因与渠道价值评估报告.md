分析背景
本数据集包含用户行为、消费、营销触达等多维信息。通过初步数据探索，我发现平台可能存在以下三个潜在优化机会：

优惠券使用效果在不同收入群体间差异明显，或存在预算浪费。

各推荐渠道的用户转化率和长期价值分布不均，渠道预算或有优化空间。

高收入用户购买转化率偏低，其行为特征与普通用户存在显著差异，值得针对性运营。
电商用户购买行为分析（第一阶段）
1. 分析背景与目标
电商平台通过发放优惠券刺激用户消费，但优惠券的实际效果需要数据验证。本次分析旨在探究：

优惠券使用次数是否对用户购买数量有显著影响？

这种影响是否因用户收入水平而异？

分析结论将用于优化优惠券发放策略，提升营销 ROI。

2. 数据说明
数据来源：模拟电商客户数据集（50万条记录）。

关键字段：

NumberOfPurchases：购买数量（目标变量）

DiscountsAvailed：优惠券使用次数（核心自变量）

AnnualIncome：年收入（潜在混淆变量）

Age、TimeSpentOnWebsite：其他可能影响购买行为的因素

数据清洗：经检查，数据无缺失值，无需额外处理。

3. 分析流程与方法
探索性分析：将用户按收入水平和优惠券使用频次分组，绘制热力图，直观观察各组平均购买数量的差异。

简单线性回归：以优惠券次数为自变量，购买数量为因变量，初步检验线性关系。

多元回归：加入收入、年龄、网站停留时间等控制变量，排除混淆因素影响。

分层回归：按收入低、中、高三组分别进行简单回归，检验效果的异质性。

4. 分析结果
4.1 热力图（收入 × 优惠券频次）
将用户按收入（低<60k、中60k-100k、高>100k）和优惠券使用频次（0次、1-2次、3-5次、6-10次）分组，计算每组平均购买数量，结果如下：

收入组	0次券	1-2次券	3-5次券	6-10次券
Low (<60k)	—	11.37	11.41	11.40
Mid (60k-100k)	—	11.37	11.38	11.37
High (>100k)	—	11.42	11.37	11.38
观察：所有格子的数值均在 11.4 左右，无明显差异，初步表明优惠券使用次数与购买数量无强关联。

4.2 简单线性回归
模型：NumberOfPurchases = β₀ + β₁·DiscountsAvailed + ε

变量	系数	标准误	t值	p值
const	11.3858	0.017	686.69	0.000
DiscountsAvailed	0.0006	0.005	0.124	0.901
优惠券次数系数为 0.0006，表示每多用一张券，平均购买数量仅增加 0.0006 件，几乎为零。

p=0.901 > 0.05，统计上不显著。

R² ≈ 0，说明优惠券次数无法解释购买数量的变化。

4.3 多元回归（控制混淆变量）
加入 AnnualIncome、Age、TimeSpentOnWebsite 后：

变量	系数	p值
const	11.3706	0.000
DiscountsAvailed	0.0006	0.900
AnnualIncome	-1.40e-08	0.948
Age	0.0005	0.314
TimeSpentOnWebsite	-0.0002	0.612
控制收入、年龄、网站时长后，优惠券系数仍为 0.0006，p=0.900，不显著。

各控制变量系数均不显著，表明购买数量的变化与这些常见因素也无关。

4.4 分层回归（按收入组）
分别对低收入、中等收入、高收入组进行简单回归：

收入组	优惠券系数	p值
Low (<60k)	0.0086	0.2845
Mid (60k-100k)	0.0034	0.6792
High (>100k)	-0.0085	0.2491
三组系数均接近 0，且 p 值均 >0.05，无统计显著性。

优惠券效果在不同收入层中均不存在，且系数正负波动，进一步印证无实际影响。

5. 结论与建议
5.1 结论
优惠券使用次数与用户购买数量之间无显著线性关系，无论是否控制收入、年龄等混淆因素，结论保持一致。

不同收入水平的用户群体中，优惠券均未表现出显著效果。

5.2 业务建议
优化发券策略：当前广撒网式的优惠券对提升购买量无效，建议调整为更高价值的定向投放（如针对高潜力用户、特定商品）。

设计阶梯优惠：可尝试“满减”或“多件折扣”等阶梯式优惠，可能更有效激发消费。

开展 A/B 测试：对新策略进行小范围实验，对比购买转化率，验证效果后再全量推广。

拓展分析维度：可进一步分析优惠券对“是否购买”（转化率）的影响，或许存在拉新价值。
项目名称
电商用户行为分析：优惠券效果归因与渠道价值评估

项目背景
某电商平台希望评估当前优惠券策略和推荐渠道的效果，以优化营销预算分配。通过分析50万条用户行为数据，探究优惠券使用次数对购买数量的影响，并比较不同推荐渠道的用户价值。

数据说明
数据来源：模拟电商客户数据集（50万条记录）

关键字段：用户ID、优惠券使用次数、购买数量、购买状态、年收入、推荐渠道、年龄、网站停留时间

分析过程与方法
数据清洗与预处理：处理缺失值（无缺失），创建收入分组和优惠券使用频次分组。

优惠券效果归因：

热力图：按收入组和优惠券频次分组，观察平均购买数量差异。

简单线性回归：评估优惠券次数与购买数量的关系。

多元回归：控制收入、年龄、网站停留时间后，检验优惠券效果的稳健性。

分层回归：分别对低、中、高收入组进行回归，检验异质性。

推荐渠道价值评估：

分组统计：计算各渠道的购买转化率、平均购买数量、平均年收入。

可视化：柱状图对比各渠道指标。

方差分析（ANOVA）：检验渠道间转化率差异是否显著。

主要发现
优惠券效果：无论是否控制收入、年龄等变量，优惠券使用次数对购买数量的影响均不显著（p=0.901，系数≈0），且在不同收入组中结果一致。表明当前优惠券策略无效。

渠道价值：各推荐渠道的购买转化率（41.6%-42.5%）、平均购买数量（11.36-11.40）、平均年收入（8.5万美元左右）高度接近。ANOVA检验显示统计显著（p=0.0426），但实际差异不足1个百分点，业务上无意义。

业务建议
优化优惠券策略：停止广撒网式发券，改为定向投放（如高潜力用户、特定商品），提高券面价值或设置阶梯优惠，并通过A/B测试验证效果。

渠道预算分配：各渠道效果均衡，建议按渠道成本或历史表现分配预算，或加强品牌一致性建设。

Python (Pandas, NumPy, Matplotlib, Seaborn, Statsmodels, SciPy)

分析方法：线性回归、多元回归、分层回归、方差分析（ANOVA）、数据可视化

独立完成从数据清洗、探索性分析、统计建模到业务建议的全流程。

客观得出“策略无效”的结论，体现严谨的数据分析思维。
